Carrier
Platform

Piattaforma IoT per la prenotazione di spedizioni e il monitoraggio in tempo reale dei viaggi di spedizione.

// 01 — Introduzione

Il progetto

Il progetto prevede una piattaforma Iot per un'azienda logistica. Permette ai suoi clienti di spedire i propri pacchi. Scegliendo di spedire il proprio pacco lungo rotte già programmate è possibile far risparmiare denaro al cliente e allo stesso tempo ridurre l'impatto ambientale.

Inoltre, è presente un sistema di monitoraggio in tempo reale con anomaly detection delle condizioni delle celle frigorifere dei veicoli in cammino.

// 02 — Funzionalità

Cosa può fare

Il sistema offre una serie di funzionalità chiave, tra cui:

Invio di un pacco

Inserendo i dati richiesti, il sistema è capace di cercare i veicoli liberi con i quali inizializzare un nuovo viaggio. È in grado anche di proporre veicoli a cui è già stato assegnato un viaggio con ancora spazio disponibile. Ciò permette l'aggregazione dei pacchi in una singola spedizione permettendo un risparmio al cliente e riducendo l'impatto ambientale.

Login

Il manager della piattaforma può accedere al sistema effettuando il login con le sue credenziali.

Aggiunta veicolo

Il manager può aggiungere nuovi veicoli al sistema, specificando i parametri richiesti.

Lista veicoli

Il manager può visualizzare la lista di tutti i veicoli registrati nel sistema.

Lista viaggi

Il manager può visualizzare la lista di tutti i viaggi programmati

Dashboard

Il manager può visualizzare una dashboard con informazioni riguardo le condizioni delle celle frigorifere dei veicoli in cammino.

Anomalie

Il manager può ricevere notifiche in tempo reale riguardo l'eventuale presenza di anomalie nei veicoli in viaggio.
